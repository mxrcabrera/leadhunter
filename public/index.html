Como obtener credenciales

API Key:

  1. Ir a console.cloud.google.com
  2. Crear proyecto o seleccionar existente
  3. Buscar y habilitar "Custom Search API"
  4. Ir a Credentials - Create Credentials - API Key

Search Engine ID:

  1. Ir a programmablesearchengine.google.com
  2. Crear motor de busqueda - "Buscar en toda la web"
  3. Copiar el Search Engine ID
